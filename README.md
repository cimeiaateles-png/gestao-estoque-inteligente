# Gest-o-de-Estoque-Inteligente
Sistema de Gestão de Estoque com persistência local (Web Storage API). Desenvolvido em JavaScript Vanilla para controle de inventário, cálculo de capital imobilizado e monitoramento de giro de estoque com interface responsiva.


📊 Gestão de Estoque Inteligente | Portfólio ADS
Este projeto foi concebido para transformar o controle de inventário manual em uma aplicação digital dinâmica. O foco principal foi a criação de uma ferramenta de apoio à decisão, que converte dados brutos em indicadores estratégicos para o microempreendedor.

🛠️ Decisões Técnicas e Arquitetura
Manipulação de DOM e Lógica CRUD: Desenvolvido integralmente em JavaScript Vanilla. A escolha por não utilizar frameworks nesta etapa foi estratégica, visando consolidar o domínio sobre o ciclo de vida dos elementos e a lógica de manipulação de objetos e arrays em tempo real.

Persistência com LocalStorage (Offline-First): Implementei o armazenamento de dados via Web Storage API. A decisão técnica garante a integridade das informações mesmo em ambientes com conexão instável, tratando o navegador como um banco de dados local persistente.

Engenharia de Interface e UX:

CSS Moderno: Uso de CSS Variables para garantir manutenibilidade e escalabilidade do código.

Layout Responsivo: Construído com Flexbox e Media Queries, assegurando uma experiência fluida tanto em desktop quanto em dispositivos móveis (Mobile-First).

Dashboard de Indicadores: Implementação de lógica para cálculo automático de Capital Imobilizado e Giro de Estoque, entregando valor analítico imediato ao usuário.

📈 Roadmap de Evolução
O projeto encontra-se na Versão 1.0 (State Local). O próximo marco técnico definido é a migração da camada de dados para uma API REST ou BaaS (Firebase), permitindo a sincronização em nuvem e o suporte a múltiplos usuários.
